Overview

This project is an interactive School Marketing Dashboard built in Power BI to analyze student acquisition, marketing performance, and engagement trends.

It helps stakeholders make data-driven decisions by identifying:

High-performing marketing channels
Conversion bottlenecks
Student demographic patterns
Engagement and retention insights
🎯 Business Problem

Educational institutions often struggle to:

Track effectiveness of marketing campaigns
Understand where leads drop in the funnel
Identify which student segments convert better

This dashboard solves these problems by providing a centralized analytical view.
